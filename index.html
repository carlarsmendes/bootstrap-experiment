Starting on the HTML Ropes

The Basics of the HTML History

  • Created by Tim Berners-Lee, 1989/1990
  • HyperText Markup Language - HTML
  • Uniform Resource Identifier - URI/URL
  • Hypertext Transfer Protocol - HTTP

Different Types of Tags

  • Content HTML Tags
  • Structural HTML Tags
  • Meta Tags (Content your PC sees, you don't)

Frequently used Content HTML Tags

  • h1-h6 - Headlines ("h1" defines the most important heading and "h6" defines the least important heading)
  • p - Paragraphs
  • strong - Bold
  • em - Italic
  • br - Line Break
  • a + href - Defines a link More Here
  • address - Defines the contact information for the author/owner of a document or an articleMore Here
    • W3Schools - HTML Basics

Frequently used Structural HTML Tags

  • html5
  • main - The site's main area
  • nav - Navigation
  • aside - opens a side bar
  • article - specifies independent, self-contained content
  • footer - Defines the footer
  • section - Defines a different section of the page
  • div - Define sections of the website; "Wrappers" of related content
    • W3Schools - HTML Basics

Frequently used Meta HTML Tags

  • !doctype html - tells the computer the kind of document it’s looking at (in this case, it’s an HTML document)
  • html
  • head & body
  • title - Although not technically a meta tag, the title tag specifies the title of your site (which is displayed on the browser tab)

    • Skillcrush - Meta Tags

Essentials on Images

  • The image tag includes src="", alt="", and width/height
  • Your source can be somewhere in the web - another server - or another folder, which is very common
  • Alt is the alternative text, usefol for accessibility, in case the browser doesn't load the image and SEO practices
  • Closing tag is />
  • Formats supported by most browsers are: jpg, png, gif and bmp and less than 1Mb
  • To have an image as a link you should write -> "< a href="" target="_blank" > "here goes your link" < / a >"
  • You can configure your images in the "head" esction, under "style"
    • W3Schools - Image Basics
