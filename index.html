Bootstrap Alerts

Bootstrap provides an easy way to create predefined alert messages. They each have their corresponding colors, just as the tables above, and can have manny effects, links, they can have an option to close, and can even incorporate animation.

This will work by incorporating several classes related to the type of alert you're looking for. Below we can find examples which incorporate the different alert types.

Success! Indicates a successful or positive action. .alert .alert-success
× Info! Indicates a neutral informative change or action. .alert .alert-info. You can click on the X symbol to close it. This uses data-dismiss aria-label inside a link. You can get more info here
Warning! Indicates a warning that might need attention. .alert .alert-warning This adds a link using the alert-link class.
× Warning! Indicates a warning that might need attention. .alert .alert-danger. This is also a closable alert but this time with an animation. this uses the classes alert-dismissible and fade in.
